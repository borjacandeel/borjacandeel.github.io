# Borja Candel — Portfolio Web Profesional

Bienvenido al repositorio de mi portfolio web personal. Este proyecto ha sido diseñado desde cero para reflejar mi perfil técnico como Administrador de Sistemas y Redes (ASIR/SMR), combinando una estética *premium* y moderna con un rendimiento técnico impecable.

## 🚀 Tecnologías

El proyecto ha sido desarrollado utilizando un stack tecnológico limpio y sin dependencias externas para garantizar el máximo rendimiento y control:

- **HTML5 Semiaśntico**: Estructura limpia y accesible.
- **CSS3 Vanilla**: Moderno diseño *Liquid Glass* (estilo iOS 26), Variables CSS, Grid/Flexbox y animaciones fluidas.
- **JavaScript Vanilla (ES6+)**: Lógica liviana para interacciones dinámicas del DOM, canvas, tooltips avanzados y control de temas oscuro/claro.
- **Cero frameworks**: Sin dependencias de React, Vue, Tailwind, ni librerías adicionales.

## ✨ Características Principales

*   🌓 **Modo Oscuro / Claro**: Cambiador de temas elegante con persistencia nativa en `localStorage`.
*   💧 **Estética Liquid Glass**: Efectos avanzados de refracción, reflejos y transparencias optimizados a nivel de tarjeta, navbar y tooltips.
*   🖱️ **Tooltips Inteligentes**: Descripciones emergentes contextuales en habilidades técnicas calculadas dinámicamente para no desbordar la pantalla y siempre centradas en el elemento.
*   📜 **Scroll Animado (Reveal)**: Los elementos de la web aparecen suavemente mediante `IntersectionObserver`.
*   ⚛️ **Canvas Interactivo**: Fondo dinámico de red neuronal o constelación interactiva que reacciona a los movimientos del ratón en la sección superior.
*   📱 **100% Responsivo**: Adaptación perfecta a móviles, tablets y monitores ultrawide.

## 🗂 Estructura de Secciones

El portfolio está divivido narrativamente para presentar rápidamente mi información profesional:

1.  **Hero Section**: Presentación de impacto con texto autotipado (Typed effect), orbes con efecto parallax, canvas en movimiento y botones directos (Descargar CV / Contacto).
2.  **Sobre Mí y Estadísticas**: Un breve resumen de mi vocación técnica junto a un contador numérico dinámico de estadísticas clave.
3.  **Habilidades Técnicas (*Skills*)**: Clasificación visual en tarjetas de Sistemas Operativos, Redes, Programación y Ciberseguridad/Virtualización.
4.  **Experiencia Profesional**: Formato de "línea temporal" (Timeline) interactiva con listas desplegables estilo acordeón para detallar las tareas realizadas en cada experiencia.
5.  **Educación**: Titulaciones en curso y finalizadas expuestas con limpieza visual.
6.  **Más Allá del Código (Intereses)**: Muestra mi perfil polivalente a través de la pasión por el ciclismo y la producción musical.
7.  **Contacto**: Accesos directos para enviar email, llamarme o descargar el currículum.
